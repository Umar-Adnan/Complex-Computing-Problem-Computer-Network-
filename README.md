# 🌐 Inter-Company Network Configuration & Subnet Optimization

A Cisco Packet Tracer implementation of a multi-company enterprise network designed for efficient IP address management, scalable subnetting, VLAN segmentation, DHCP/DNS services, RIP, and BGP routing.

## Overview

This project simulates four interconnected companies (JTECH, UCOM, TNet, and NSys) while maintaining independent IP spaces and enabling secure inter-company communication.

## Features

- VLSM & CIDR subnetting
- VLAN-based segmentation
- Router-on-a-Stick
- DHCP, DNS & HTTP services
- RIP (internal routing)
- BGP (external routing)
- End-to-end connectivity validation

## Running

1. Open `Project.pkt` in Cisco Packet Tracer.
2. Verify routing tables.
3. Test connectivity using `ping`.
4. Verify DHCP and DNS functionality.
